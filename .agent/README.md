# AI Documentation Hub

This `.agent` folder serves as the central documentation hub for AI agents working on this project. It maintains up-to-date information about the codebase, processes, and implementation plans.

**Purpose**: Product Requirements Documents (PRDs) and implementation plans for features

- Implementation roadmaps and milestones
- User stories and acceptance criteria
- Technical specifications and constraints
